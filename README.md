# TODO list project
using useDark library @vueuse/core and jsonplaceholder API

state manager - VueX

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
